SOFTWARE

Software de ofimatica

conjunto de herramientas informáticas que automatizan y optimizan las tareas de oficina, como la creación de documentos, hojas de cálculo y presentaciones, así como la gestión de datos y comunicaciones

Software de diseño grafico

programa o conjunto de programas que permiten crear, editar y manipular imágenes y elementos visuales en una computadora, desde dibujos simples hasta modelos 3D complejos y animaciones, sirviendo para diversas aplicaciones como diseño web, publicidad, ilustración y autoedición

Software de programación

conjunto de herramientas que usan los desarrolladores para crear, mantener y probar otros programas, como aplicaciones y sistemas operativos. Incluye editores de texto, compiladores, depuradores y entornos de desarrollo integrados (IDE) que facilitan la escritura y ejecución de código en diferentes lenguajes de programación

Software de arquitectura

diseño fundamental de un sistema, como el plano de un edificio, que define sus componentes principales (módulos, servicios), sus interrelaciones y los principios que guían su estructura y evolución, con el fin de crear una base sólida para que el software sea escalable, mantenible y cumpla sus requisitos.

Software de animacion

herramientas digitales que permiten crear imágenes en movimiento, desde dibujos cuadro a cuadro hasta animaciones 3D complejas, mediante la manipulación de fotogramas, modelado de personajes, aplicación de texturas, edición de audio y la generación de movimientos fluidos a través de la interpolación entre fotogramas clave

Software de contabilidad

programa informático que automatiza y simplifica el registro, procesamiento y análisis de toda la información financiera de una empresa, permitiendo controlar ingresos, gastos, facturación, inventarios, nóminas y generar estados financieros (como balances y estados de resultados) de forma más rápida, precisa y organizada

HARDWARE

Hardware interno

componentes físicos esenciales que están dentro de la carcasa (torre) de una computadora, como la placa base, el procesador (CPU), la memoria RAM, el disco duro (HDD/SSD), la tarjeta gráfica y la fuente de alimentación

Hardware externo

todos los componentes físicos que se conectan a una computadora (o dispositivo principal) para expandir sus capacidades o mejorar su funcionalidad, pero que no forman parte de su núcleo

Hardware de almacenamiento

componentes físicos de una computadora (como discos duros, SSDs, memorias USB, discos externos) que permiten guardar, leer y recuperar información digital (documentos, fotos, videos, programas) de forma permanente o temporal para su uso futuro, siendo fundamentales para la capacidad y velocidad del sistema
